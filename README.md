# 📊 Financial Analysis & Company Insight Platform

A full-stack financial analysis platform developed during my Software Development Engineer (SDE) Internship at **Bluestock Fintech Pvt Ltd**.

---

## Overview

This application fetches financial data for publicly listed companies, processes and analyzes key financial metrics, stores the processed results in MySQL, and displays interactive insights through a modern React dashboard.

---

## Features

- Financial data fetching using external APIs
- Data cleaning and preprocessing
- Rule-based financial analysis
- Automated Pros & Cons generation
- REST API using Flask
- MySQL database integration
- Interactive React dashboard
- Redux Toolkit state management
- Tailwind CSS responsive UI
- Chart.js visualizations
- Company comparison
- Dark mode support
- CLI logging with colored output
